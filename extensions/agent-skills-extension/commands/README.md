# Custom Commands

This directory contains custom commands bundled with the extension.

## activate-skill

**Description**: Activate a skill interactively

**Usage**:

```bash
# Interactive mode - lists all available skills
/activate-skill

# Direct activation
/activate-skill skill-name
```

**Features**:

- Automatically lists all discovered skills
- Interactive selection dialog
- Activates the chosen skill
- Works with both workspace and user skills

**Files**:

- `activate-skill.toml` - Gemini CLI version
- `activate-skill.md` - OpenCode version
